APLIKACIJA "VRTICI"

Aplikacija za praćenje aktivnosti sportskog društva u vrtićima

Front-end: Vue.js (Composition API)

    - vue-router
    - vuex
    - vue-number-format

Back-end:

    - Node.js
    - Express.js,
    - ORM: Sequelize.js,
    - JsonWebToken,
    - bcrypt.js,
    - dotEnv.js,
    - express-validator
    - MySQL
